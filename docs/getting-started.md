# Stock Master 新手快速上手指南

> 从零开始，5 分钟学会用 AI 辅助你的股票投资研究。

---

## 安装与配置

### 1. 安装 Stock Master

```bash
pip install -e .
```

安装完成后，你就可以在终端里使用 `sm` 命令了。试试看：

```bash
sm --help
```

### 2. 环境变量（可选）

Stock Master 默认使用免费数据源（AkShare），**不需要任何配置就能用**。

如果你有付费数据源的 Token，可以设置以获取更完整的数据：

| 环境变量 | 说明 | 是否必需 |
|----------|------|----------|
| `SM_IFIND_TOKEN` | 同花顺 iFinD 付费数据 | 可选 |
| `SM_CHOICE_TOKEN` | 东方财富 Choice 付费数据 | 可选 |
| `SM_WIND_TOKEN` | Wind 付费数据 | 可选 |

没有这些 Token 也完全没关系——系统会自动降级到免费数据源，核心功能不受影响。

## 你的第一次调研（5 分钟上手）

让我们以「水晶光电（002273）」为例，走一遍完整流程。

### 第 1 步：准备调研

```bash
sm research 002273
```

这条命令会自动：
- 从网上拉取该股票的最新行情、财务、估值、新闻等数据
- 生成一个结构化的 `context.md`（数据上下文包）和 `dossier.yaml`（事实档案）
- 在 `research/002273/` 下创建完整的调研目录

你会看到类似这样的输出：

```
调研目录已准备就绪：research/002273/2026-04-03

文件结构：
  📄 context.md     — 数据上下文包
  📄 dossier.yaml   — 结构化事实包与新手行动建议
  📁 agents/         — 各角色调研报告待填入
  📄 stock-report.md — 一页式结论页（8 个关键问题）
  📄 synthesis.md    — 综合研判（待生成）
  📄 decision.md     — 决策模板（待你填写）
```

### 第 2 步：在 Cursor 中查看调研材料

打开 Cursor，浏览刚才生成的文件：

- **`context.md`** — 这是 AI 需要的原始数据包，包括行情走势、估值指标、财务数据、新闻摘要。里面还附带了指标解读（比如 PE 是什么意思），适合新手学习。
- **`dossier.yaml`** — 这是结构化事实档案，会告诉你「证据覆盖度」（数据收集了多全面）和「新手行动建议」（该买还是先观望）。

> 💡 **小贴士**：先看 `dossier.yaml` 里的 `rookie_action` 部分——它会用简单的语言告诉你这只股票当前适合什么操作。

### 第 3 步：查看评分

```bash
sm score 002273
```

系统会从 5 个维度给出评分：

| 维度 | 权重 | 含义 |
|------|------|------|
| 质量 | 28% | 公司的盈利能力和财务健康程度 |
| 估值 | 22% | 当前价格是贵了还是便宜了 |
| 趋势 | 20% | 股价的走势和动量 |
| 风险 | 20% | 潜在的下行风险有多大 |
| 催化剂 | 10% | 有没有可能推动股价上涨的事件 |

每个维度 0~100 分，还会给出一个「可信度」——告诉你这个评分的数据依据有多充分。

### 第 4 步：买前检查

如果你觉得这只股票不错，想买入 5% 的仓位，先做个检查：

```bash
sm check-buy 002273 --position-pct 5
```

系统会综合检查：
- 你的组合是否会过于集中在某个行业
- 当前价格是否在追高
- 有没有必要的研究证据缺失
- dossier 给出的建议仓位上限是多少

### 第 5 步：加入观察清单

暂时不想买？先加入观察清单跟踪：

```bash
sm watchlist --action add --code 002273 --bucket ready
```

分组说明：
- `ready` — 已研究充分，等待合适时机
- `wait_price` — 等目标价位到达
- `avoid` — 已分析过，不适合买入

## 日常工作流

### 每日：查看提醒

```bash
sm alerts --scan
```

系统会扫描你的持仓和观察清单，检测 8 类异动：
- 重大公告（增持、回购、并购等）
- 业绩预告变动
- 资金异常流入/流出
- 估值到达异常区间
- 目标价触发
- 风险预警
- 需要再平衡
- 回撤超阈值

看完提醒后，用 `sm alerts --ack 0` 确认已处理的提醒（数字是提醒的序号）。

### 研究新股

```bash
sm research 600519
sm score 600519
sm check-buy 600519 --position-pct 3
```

流程是固定的：**调研 → 评分 → 买前检查 → 决策**。

### 记录交易

当你在真实账户中完成了交易，记得记录下来：

```bash
sm trade buy 002273 --price 20.5 --shares 500 --reason "估值合理，行业回暖"
```

系统会自动：
- 生成交易记录（YAML 格式）
- 更新持仓状态
- 创建叙事日志（方便日后复盘）

### 查看持仓

```bash
sm portfolio
```

### 周末复盘

每周花 15 分钟回顾本周操作：

```bash
sm weekly-review
```

系统会生成一份复盘模板，并自动附上「行为偏差提示」——比如提醒你「本周交易 5 次，频率偏高」或「有一笔交易在高位追入」。

### 进阶：多模型综合建议

```bash
sm suggest
```

这会调用多个 AI 模型（GPT/Claude/Gemini）分别给出独立建议，然后生成一份综合报告。结果保存在 `research/_suggest/` 目录下。

> ⚠️ `sm suggest` 需要先登录 Cursor Agent CLI，首次使用请运行 `sm agent-login`。

## 模拟盘练手

**强烈建议在投入真金白银之前，先用模拟盘练习。**

模拟盘初始资金 100 万，所有操作不涉及真实资金。

### 买入

```bash
sm paper-trade buy --code 002273 --price 20.5 --shares 500
```

### 卖出

```bash
sm paper-trade sell --code 002273 --price 22.0 --shares 500
```

### 查看持仓

```bash
sm paper-trade status
```

### 查看业绩

```bash
sm paper-trade performance
```

会显示：初始资金、当前现金、持仓市值、总资产、总收益率、交易次数。

> 💡 模拟盘的数据保存在 `journal/paper_portfolio.yaml`，你可以随时查看历史记录。

## 学习成长

Stock Master 不仅是工具，更是你的投资学习伙伴。

### 查看行为画像

```bash
sm profile
```

系统会分析你的全部交易记录，告诉你：
- **总交易次数和胜率** — 你赢了多少次？
- **平均持有天数** — 你是短线还是中线风格？
- **优势** — 你做对了什么？
- **常见错误** — 你容易在哪里犯错？
- **行为偏差** — 有没有「追涨杀跌」「过度交易」等问题？
- **学习建议** — 接下来应该改进什么？

### 积累个人规则

系统会在 `journal/personal_rules.yaml` 中自动积累你的个人交易规则。这些规则来自你的实际交易经验——比如「某行业不适合重仓」或「高估值股需要更严格的止损」。

随着交易次数增加，这个规则库会越来越有价值。

### 坚持复盘

每次交易后，打开 `journal/entries/` 下的叙事日志，补充：
- 当时的决策逻辑是什么？
- 情绪状态如何？
- 如果重新来过，会做什么不同的决定？

这些记录会帮助你更快成长。

## 常见问题 FAQ

### 如何理解评分？

评分范围 0~100，从 5 个维度综合评估一只股票：
- **60 分以上** 通常表示该维度表现不错
- **40 分以下** 需要关注，可能有问题
- **综合分** 是加权后的总分，但不能仅凭一个数字做决策

评分是辅助工具，不是买卖信号。它帮你快速了解一只股票的大致情况，真正的决策需要结合调研报告和你自己的判断。

### 什么是可信度？

可信度表示评分的「数据基础有多扎实」，范围 0~100：
- **80+** 数据充分，评分可参考性强
- **50~80** 部分数据缺失，评分仅供参考
- **50 以下** 数据严重不足，评分参考价值有限

如果可信度低，说明系统拿到的数据不够全面，这时候评分可能不准确。你可以接入付费数据源来提升覆盖率。

### 为什么建议先用模拟盘？

投资是一项需要练习的技能。模拟盘让你在不承担真实风险的情况下：
- 熟悉系统的所有功能
- 练习完整的调研 → 决策 → 交易 → 复盘流程
- 发现自己的行为偏差
- 建立自己的交易纪律

当你在模拟盘中稳定盈利、并且形成了自己的投资规则之后，再考虑用真金白银。

### 数据有缺失怎么办？

系统会在 `dossier.yaml` 中明确标注哪些数据缺失（`missing_sections`）。常见解决方案：

1. **等待更新** — 某些数据（如季报）可能还没发布
2. **接入付费数据源** — 设置 `SM_IFIND_TOKEN` 等环境变量，获取更全面的数据
3. **手动补充** — 在 Cursor 中用联网搜索补充缺失信息
4. **关注可信度** — 如果缺失数据较多，降低对评分的依赖，更多依赖定性分析

数据缺失不代表不能分析，只是需要你在决策时更加谨慎。

### 持仓截图功能怎么用？

如果你想从券商 App 的持仓截图自动更新持仓数据：

```bash
sm snapshot 截图.png
```

系统会用 AI 识别截图中的持仓信息，自动更新 `journal/portfolio.yaml`。首次使用需要先登录：

```bash
sm agent-login
```
